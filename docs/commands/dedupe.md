# dedupe

Find and remove duplicate media files.

## Usage

```bash
exifutils dedupe <FILE|DIR>...
```

## Description

Scans the specified files or directories for duplicate media files and helps you remove them.

## Examples

```bash
exifutils dedupe /path/to/photos/
```

## Notes

For advanced deduplication needs, consider dedicated tools like:
- [fdupes](https://github.com/adrianlopezroche/fdupes)
- [rdfind](https://github.com/pauldreik/rdfind)
- [jdupes](https://github.com/jbruchon/jdupes)
